A modern web platform for instant freight quotes, real-time shipment tracking, and automated customs documentation.

## Features
- ✅ Multi-user authentication (Shipper, Forwarder, Admin)
- ✅ 3-step onboarding flow
- ✅ Role-based dashboards
- ✅ Instant freight quotes marketplace
- ✅ Real-time shipment tracking
- ✅ Automated customs document generation
- ✅ Escrow payment management
- ✅ Cargo insurance integration
- ✅ OFAC compliance screening
